# Syllabus — Analysis and Design of Algorithms

## UNIT-I — Introduction
- Algorithm analysis: Time and Space Complexity
- Asymptotic Notations and properties
- Best case, Worst case, Average case analysis
- Recurrence relation: Substitution method
- Lower bounds
- Linear Search, Binary Search, Interpolation Search
- Pattern Search: Naive string-matching algorithm
- Rabin-Karp algorithm
- Knuth-Morris-Pratt algorithm
- Sorting: Insertion sort, Heap sort

## UNIT-II — Graph Algorithms
- Representations of graphs
- Graph traversal: DFS and BFS applications
- Connectivity, Strong connectivity, Bi-connectivity
- Minimum Spanning Tree: Kruskal's and Prim's
- Shortest path: Bellman-Ford algorithm
- Dijkstra's algorithm
- Floyd-Warshall algorithm
- Network flow: Flow networks, Ford-Fulkerson method
- Maximum bipartite matching

## UNIT-III — Algorithm Design Techniques
- Divide and Conquer: Finding maximum and minimum
- Merge sort, Quick sort
- Dynamic Programming: Elements
- Matrix-chain multiplication
- Multi stage graph
- Optimal Binary Search Trees
- Greedy Technique: Elements
- Activity-selection problem
- Optimal Merge pattern
- Huffman Trees

## UNIT-IV — State Space Search Algorithms
- Backtracking: N-Queens problem
- Hamiltonian Circuit Problem
- Subset Sum Problem
- Graph colouring problem
- Branch and Bound: 15-Puzzle problem
- Assignment problem
- Knapsack Problem
- Travelling Salesman Problem

## UNIT-V — NP-Complete and Approximation
- Tractable and intractable problems
- Polynomial time algorithms
- NP-algorithms
- NP-hardness and NP-completeness
- Bin Packing problem
- Problem reduction: TSP, 3-CNF
- Approximation Algorithms
- Randomized Algorithms: primality testing
- TSP, Randomized quick sort
- Finding kth smallest number
